Operadores e funções

  • = e != — igualdade (texto sem diferenciar maiúsculas)
  • ~ — contém (busca parcial)
  • IN ( "A", "B" ) e NOT IN — apenas para project
  • IS EMPTY / IS NOT EMPTY — campos opcionais (ver tabela)
  • >, <, >=, <= — datas em formato ISO-8601
  • me() e currentUser() — usuário autenticado em assignee ou author

Ordenação

ORDER BY aceita: created, updated, finished, title, identifier, priority.
